## Overview
This project is an open-source investigation into the expansion of mining activity in Afghanistan following the Taliban’s return to power.

Using satellite imagery and publicly available data, the project analyses changes in mining infrastructure between 2019 and 2023. It combines geospatial analysis with open-source reporting to identify patterns of development and potential economic activity in resource-rich regions.

---

## Objectives
- Track changes in mining sites over time  
- Identify expansion of infrastructure using satellite imagery  
- Combine remote sensing data with open-source reporting  
- Present findings through interactive visualisation  

---

## Methodology

### 1. Satellite Imagery Analysis
- Google Earth Engine was used to analyse satellite data  
- Imagery from multiple years (2019–2023) was compared  
- Band combinations were applied to highlight land-use changes  

### 2. Geolocation & Chronolocation
- Specific mining sites were identified and tracked over time  
- Changes in terrain and infrastructure were analysed to establish timelines  

### 3. Open-Source Research (OSINT)
- Findings were cross-referenced with reporting from international media outlets  
- Publicly available information was used to contextualise developments  

---

## Tools & Technologies
- Google Earth Engine (JavaScript)  
- HTML, CSS, JavaScript

- Open-source intelligence (OSINT) techniques

## Key Findings
- Evidence suggests expansion of mining infrastructure in selected regions  
- Increased activity aligns with shifts in political control and economic priorities  
- Satellite imagery reveals patterns not always visible in traditional reporting  

---

## 💡 Significance
This project demonstrates how open-source tools and satellite data can be used to investigate complex geopolitical and economic developments in inaccessible regions.
